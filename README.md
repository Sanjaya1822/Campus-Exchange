# Campus Exchange

Campus Exchange is a web-based application designed for college students to Buy, Sell, Rent, or Donate items within the campus.  
The platform helps students exchange books, electronics, accessories, and other essentials in a secure and organized manner.

---

##  Objective

The main objective of Campus Exchange is to:
- Reduce waste by promoting reuse
- Help students save money
- Provide a safe, campus-only marketplace
- Encourage peer-to-peer interaction

---

##  Features

-  User registration and login
-  Post items for Buy / Sell / Rent / Donate
-  Browse and search available items
-  Edit and delete item listings
-  Contact seller/donor
-  Simple and user-friendly interface
